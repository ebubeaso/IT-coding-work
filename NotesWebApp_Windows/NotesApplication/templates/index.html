Welcome to my Flask Notes application!


The purpose of this application:

I am making this because I wanted to get better at making webpages using Flask. I really enjoyed my first API application with querying employees (I will continue to add to it) on the web UI and then more data management on using API tools like Postman and Python requests to communicate with it. Now, I decided to continue to practice my skills to make a notes application where you can see the notes that you have made!

You will be able to add notes from the site by first logging in (or signing up) and then going to the notes section, where you can add, modify or delete notes from there. However, you would need to have a registered account to access the Notes page.

Instructions

To add a note: click on 'Make a New Note' button, and a popup will come up. In the popup, enter the new note that you would like to add in the text box. Once you are done, click on the 'Add Note' button. If you are stuck on a page that says 'Redirecting...', simply click on the blue link on the page to go back onto the notes page.

To modify a note: click on the 'Modify' button that is right next to the note that you would like to modify. Once you do, a text box will pop up, and you will just enter in a new note. Once you are done, click on 'Update Note'. A popup will appear confirming the note was updated. Click 'OK' and you will be redirected to the notes page.

To delete a note: click on the 'Delete' button that is right next to the note that you would like to remove. Once you do, a popup will show, confirming if you would like to make the change. Click 'OK' and you will receive an alert that confirms that you deleted the note. Click 'OK' and you will be redirected to the notes page.

That is how you use this notes application! Click here to login to the notes application. You will also be able to sign up to make a new account. Be advised that you need an account to use the Notes section!!
